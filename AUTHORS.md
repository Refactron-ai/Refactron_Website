## How to Add Yourself

If you've contributed to this project and would like to add yourself to this list:

1. Add your name in alphabetical order by last name
2. Optionally include your GitHub username
3. Optionally include your email or website

Format: `- Full Name (@github-username) - [optional contribution description]`

Example:
- John Doe (@johndoe) - Initial documentation setup
- Jane Smith (@janesmith) - Bug fixes and UI improvements
